# SprintDesk 

SprintDesk is a full-stack collaborative project management platform designed to streamline project planning, issue tracking, team communication, and subscription management in a unified environment.
 
##  Features
 
-  JWT-based Authentication & Authorization
-  Project Creation and Management
-  Issue Tracking and Assignment
-  Project Chat & Issue Comments
-  Team Collaboration via Invitations
-  Subscription & Payment Integration
-  Project Progress Monitoring
-  Search and Filter Functionality
 
##  Tech Stack
 
### Frontend
- React.js
- Redux Toolkit
- Vite
- Tailwind CSS
- Axios
 
### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL
 
### Other Integrations
- Razorpay Payment Gateway
- REST APIs
- WebSocket Ready Architecture
